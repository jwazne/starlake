## Download

To install the starlake CLI, you need to download the starlake script from github. 
The script will in turn download required dependencies and copy them to the bin subdirectory.

<Tabs groupId="platforms">
<TabItem value="linux_macos" label="Linux/MacOS">

```sh
$ mkdir $HOME/starlake
$ cd $HOME/starlake
$ curl -O hhttps://raw.githubusercontent.com/starlake-ai/starlake/master/distrib/starlake.sh
$ chmod +x starlake.sh
$ ./starlake.sh
```

</TabItem>
<TabItem value="windows" label="Windows">

```powershell
c:\> mkdir %userprofile%\starlake
c:\users\me\starlake> cd %userprofile%\starlake
c:\users\me\starlake> curl -O https://raw.githubusercontent.com/starlake-ai/starlake/master/distrib/starlake.cmd
c:\users\me\starlake> starlake
```

</TabItem>
</Tabs>

:::note

Any library you may need (Snowflake or Oracle client)  wille need to copied in the bin/spar/jars folder.

:::

## Docker

You may also run Starlake from a docker container. To do so, download this [Dockerfile](https://github.com/starlake-ai/starlake/blob/master/distrib/Dockerfile)
and build your docker image


<Tabs groupId="platforms">
<TabItem value="linux_macos" label="Linux/MacOS">

```sh
$ mkdir starlake
$ cd starlake
$ curl -O hhttps://raw.githubusercontent.com/starlake-ai/starlake/master/distrib/starlake.sh
$ curl -O https://raw.githubusercontent.com/starlake-ai/starlake/master/distrib/Dockerfile
$ docker build --build-arg STARLAKE_VERSION=0.5.3 -t starlake .
$ docker run -it starlake
```

</TabItem>
<TabItem value="windows" label="Windows">

```powershell
c:\> mkdir starlake
c:\starlake> cd starlake
c:\starlake> curl -O https://raw.githubusercontent.com/starlake-ai/starlake/master/distrib/starlake.cmd
c:\starlake> curl -O https://raw.githubusercontent.com/starlake-ai/starlake/master/distrib/Dockerfile
c:\starlake> docker build --build-arg STARLAKE_VERSION=0.5.3 -t starlake .
c:\starlake> docker run -it starlake
```

</TabItem>
</Tabs>

## Specific Version

To install a specific version (0.5.3 in the example below), set the COMET_VERSION environment variable before running starlake:

<Tabs groupId="platforms">
<TabItem value="linux_macos" label="Linux/MacOS">

```sh
$ COMET_VERSION=0.5.3 ./starlake.sh
```

</TabItem>
<TabItem value="windows" label="Windows">

```powershell
c:\starlake> SET COMET_VERSION=0.5.3
    c:\starlake> starlake

```

</TabItem>
</Tabs>
