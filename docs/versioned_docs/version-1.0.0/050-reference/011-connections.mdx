# Connections

Connections are defined in the `connections` section under the root attribute `application`.


The following types of connections are supported:

- Spark / Databricks (see [Spark](#spark))
- JDBC using  (see [Snowflake](#snowflake))
- Redshift (see [Redshift](#redshift))
- BigQuery (see [BigQuery](#bigquery))
- Postgres (see [Postgres](#postgres))
- MySQL (see [MySQL](#mysql))
- Oracle (see [Oracle](#oracle))
- SQL Server (see [SQL Server](#sql-server))
- Local file system
